#### External Method
Include the stylesheet as a tag in html
```html
<link rel="stylesheet" href="./styles.css">
```

#### Internal Method
Include the styling css internally to the html
```html
<head>
 <style>
    div {
      color: white;
      background-color: black;
    }
    p {
      color: red;
    }
  </style>
</head>
```
